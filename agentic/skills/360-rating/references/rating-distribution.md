# 绩效评级分布规则

公司采用强制分布制度，各等级对应分数段与人数比例如下：

| 最终绩效结果（总分 X） | 绩效等级   | 强制比例       |
|----------------------|-----------|---------------|
| 4.9 ≤ X ≤ 5         | S  卓越   | <1%           |
| 4.5 ≤ X < 4.9       | A+ 优秀   | ≤5%           |
| 4.0 ≤ X < 4.5       | A  良好   | ≤10%          |
| 3.5 ≤ X < 4.0       | B+ 潜力   | 15%           |
| 3.0 ≤ X < 3.5       | B  合格   | 60%–65%       |
| 2.0 ≤ X < 3.0       | B- 待改进 | 5%–10%        |
| 1.0 ≤ X < 2.0       | C  不合格 | —             |

## 注意事项

- **大多数人落在 B（合格）区间**，这是正常且预期的结果。
- S、A+、A 属于稀缺名额，须有充分理由支撑高分。
- 若目标等级为 B+ 及以上，评分人需确认该候选人在团队中确实属于前 15-16%。
- 打分时应以目标等级对应的分数段为锚点，各维度分数的加权均值需落入该区间。
